## 📌 Featured Projects

### 1. CAT 924Gz Transmission Control Retrofit
**Problem:** Loader CAT 924Gz had a faulty transmission control system. The original ECU was expensive and hard to find.

**Solution:** Performed reverse engineering on the electrical system, rewired the solenoid controls based on the original **truth table**, and restored full functionality **without replacing the ECU or solenoids**.

**Tech:** Electrical schematic analysis, 24Vdc wiring, solenoid drivers

---

### 2. Star-Delta Motor Control with IoT Monitoring
**Problem:** Industrial 3-phase motors required remote monitoring to prevent downtime.

**Solution:** Designed and assembled a 3-phase 220Vac/50Hz star-delta control panel with contactors, timers, and overload relays. Integrated IoT sensors to monitor **ON/OFF status, current, temperature**, and send automatic notifications via smartphone.

**Tech:** 3-phase panel wiring, ESP32, IoT sensors, Telegram API
